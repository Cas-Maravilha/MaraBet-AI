# Instalar psycopg2 para Python

## Instalação

```bash
pip install psycopg2-binary
```

Ou usando o arquivo requirements:

```bash
pip install -r requirements_postgres.txt
```

## Versão alternativa (psycopg2 em vez de psycopg2-binary)

Se preferir a versão compilada:

```bash
pip install psycopg2
```

**Nota:** `psycopg2-binary` é mais fácil de instalar pois já inclui as bibliotecas necessárias.
